## 8. Data Quality Notes & Known Limitations

| Issue | How We Handle It | Limitation to Mention in Report |
| :--- | :--- | :--- |
| \N nulls throughout Ergast CSVs | Load with dtype=str → replace \N with pd.NA → cast columns after | None — fully handled |
| Pit stop data sparse before 2010 | All pit stop + qualifying analysis scoped to 2010–2024 only | Historical comparison pre-2010 not pit adjusted |
| 10,953 DNFs with no position | positionOrder used (always assigned 1–max). is_dnf flag handles classification. | DNF finishing order classification may vary from official records |
| No tyre compound data in Ergast | Use lap_time_std as compound bias proxy (degradation heuristic) | Compound analysis is inferred, not sourced from actual Pirelli data |
| Qualifying null rate >60% for Q2/Q3 | Use best_q_ms (best qualifying time across any session). qualifying_gap_ms = best_q - pole. | Q2/Q3 analysis only reliable for 2005+ when modern qualifying format started |
